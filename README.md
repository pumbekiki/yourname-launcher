# yourname-launcher

Современный лаунчер Minecraft для Windows с аккуратным темным интерфейсом, автоматической установкой игры, поддержкой популярных загрузчиков и встроенной системой логов.

## О проекте

`YourName Launcher` - это десктопный лаунчер, созданный для удобного запуска Minecraft без лишней возни с файлами, Java и ручной настройкой клиента.

Лаунчер умеет:

- устанавливать официальные версии Minecraft от `1.0` до актуальной последней
- работать с `Vanilla`, `Forge`, `Fabric` и `OptiFine`
- автоматически находить нужную `Java`
- скачивать подходящую версию `Java`, если ее нет на компьютере
- хранить разные сборки в отдельных папках инстансов
- показывать прогресс установки и живые логи прямо в интерфейсе
- открывать папку текущей сборки с `mods`, `resourcepacks`, `saves` и другими файлами

## Технологии

Проект собран на следующем стеке:

- `Electron` - оболочка десктопного приложения
- `HTML / CSS / Vanilla JavaScript` - интерфейс и клиентская логика
- `Python` - backend-часть лаунчера
- `minecraft-launcher-lib` - установка и запуск Minecraft
- `PyInstaller` - упаковка Python backend в `.exe`
- `electron-builder` - сборка portable-версии и Windows-инсталлятора

## Возможности

- Поддержка оффлайн-ника
- Автоопределение Java по версии Minecraft
- Автозагрузка Java при необходимости
- Отдельные папки для разных версий и загрузчиков
- Установка клиента прямо из лаунчера
- Запуск Minecraft из интерфейса
- Кнопка открытия папки текущего инстанса
- Встроенная консоль с логами лаунчера и игры
- Поддержка portable-сборки и установщика

## Структура проекта

```text
yourname-launcher/
|-- backend/
|   `-- server.py              # Установка, запуск Minecraft, Java, логика backend
|-- public/
|   |-- index.html             # Основная HTML-структура
|   `-- app.js                 # Вся клиентская логика интерфейса

|-- preload.js                 # Безопасный мост между UI и Electron

```

## Запуск для разработки

Требуется:

- `Node.js`
- `Python 3`

Установка зависимостей и запуск:

```bash
npm install
npm start
```

## Сборка

Сборка portable-версии:

```bash
npm run build:win
```

Сборка установщика:

```bash
npm run build:installer
```

## Готовые сборки

После сборки основные файлы лежат здесь:

- `release/win-unpacked` - portable-версия
- `release/YourName-Launcher-Setup-1.0.0.exe` - установщик с выбором папки установки
